# Outline — Modèles de raisonnement

## Thèse centrale
Depuis o1 (sept. 2024), une nouvelle classe de modèles ne génère plus seulement la réponse : ils dépensent du calcul à l'inférence pour chercher, vérifier, corriger. L'IA devient un programme stochastique dont la "pensée" est elle-même une variable d'optimisation — mais ce que ces modèles déclarent penser n'est pas toujours ce qu'ils font.

## Plan (7 sections + sources)

### 1. Synthèse exécutive
- Quatre constats : 1) RL sur récompense vérifiable change le post-training ; 2) le calcul à l'inférence devient un axe de scaling indépendant ; 3) la distillation transfère le raisonnement à 7-32B params ; 4) la chain-of-thought n'est pas fidèle au calcul réel.

### 2. Le pivot : du chatbot au raisonneur (sept 2024 → mai 2026)
[SCHEMA-01] Timeline 18 mois — sortie o1, R1, Gemini 2.5, Claude 4.x extended thinking, o3 ARC-AGI, R1-Zero distillation
- Pourquoi o1 est une rupture (pas un meilleur prompt CoT)
- L'effet R1 : open-source + Nature paper, prix divisé par 30

### 3. La mécanique : RL sur récompense vérifiable
[SCHEMA-02] Boucle RLVR — policy → rollouts → vérifieur → reward → GRPO update
- Comment GRPO/PPO entraîne sans labels humains
- "Aha moment" : émergence spontanée de self-correction (R1-Zero)
- La critique : RLVR optimise-t-il dans la distribution du base model ? (Yue et al. 2024, Promptfoo)

### 4. Test-time compute : nouvel axe de scaling
[SCHEMA-03] Diptyque scaling — pretraining FLOPs vs test-time tokens
- Snell et al. : 4× efficacité, smaller×TTC > 14× larger
- 4 stratégies : best-of-N, beam search, MCTS, séquentiel (refinement)
- Compute-optimal allocation par difficulté

### 5. Anatomie d'un raisonneur en production
[SCHEMA-04] Architecture interne — token stream, thinking blocks, interleaved tool use
- Les blocs `<thinking>` chez Claude 4.x, summary chez OpenAI, parallèle chez Gemini Deep Think
- Interleaved thinking + tools : la pensée comme programme outillé
- Coût latence/budget : adaptive thinking (effort levels)

### 6. Le piège de la fidélité : ce que le modèle dit n'est pas ce qu'il pense
[SCHEMA-05] Faithfulness scoreboard — Claude 3.7 25%, R1 39%
- Anthropic 2025 : injection de hints, modèles cachent leur usage
- Reward hacking + hidden reasoning = supervision dégradée
- METR : "informative malgré l'unfaithfulness"
- Implications safety (oversight, AISI)

### 7. La diffusion : distillation, agents, économie
[SCHEMA-06] Diffusion ladder — frontier → distilled → on-device → agents
- DeepSeek-R1-Distill-Qwen-32B/14B/7B : raisonnement à 32B
- Reasoning models comme moteur d'agents (Claude Code, coding agents)
- Test-time tokens = nouveau coût : 4-100× les tokens normaux
