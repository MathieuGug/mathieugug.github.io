# Expériences narratives — étude

> 5 mai 2026 — Mathieu Guglielmino

Étude illustrée sur les **expériences narratives** : data storytelling, visualisation narrative, IA générative dans la chaîne. État de l'art, fondations canoniques (Bertin, Shneiderman, Munzner, Cairo, Meirelles), tournant humaniste (Lupi, Posavec, Fragapane), architectures techniques (scrollytelling vs pipeline agentique), patterns d'usage et cadre praticien.

## Contenu du dossier

- `index.html` — hub d'entrée du dossier.
- `20260505-narrative-experiences-app.html` — application interactive (à ouvrir dans un navigateur, fonctionne en `file://`).
- `20260505-narrative-experiences-rapport.md` — rapport en markdown (Obsidian-compatible).
- `images/` — sept schémas SVG :
  1. Spectre auteur ↔ lecteur (Segel-Heer 2010, sept genres positionnés)
  2. Stack des fondations canoniques (Bertin → Shneiderman → Meirelles → Munzner → Cairo)
  3. Matrice genres × dimensions de design
  4. Manifeste Data Humanism (moderniste vs humaniste)
  5. Pipelines techniques (scrollytelling classique vs agentique IA-driven)
  6. Chaîne de production augmentée par IA (sept étapes, apport et risque)
  7. Matrice contexte × pattern (cinq cas-types)
- `images/refs/` — captures de référence embarquées dans cinq modaux à visée pédagogique (Parable of the Polygons CC0, Snow Fall NYT 2012, Dear Data, portfolio Federica Fragapane, The Pudding).

## Comment lire

- **Application interactive** : ouvrir `20260505-narrative-experiences-app.html` directement dans un navigateur. Schémas cliquables, glossaire au survol, sommaire et sources actifs, zoom plein écran sur chaque schéma, sidebar des sources réductible.
- **Markdown** : lire `20260505-narrative-experiences-rapport.md` dans Obsidian, GitHub ou VS Code. Les schémas SVG sont référencés en syntaxe Obsidian `![alt|width](images/...svg)`.
