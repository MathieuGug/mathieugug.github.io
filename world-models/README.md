# World models — comprendre les enjeux

> 5 mai 2026 — Mathieu Guglielmino

Étude approfondie sur les *world models* en IA en 2026 : qu'est-ce que c'est, pourquoi le sujet bascule en 2025-2026, panorama des dix acteurs majeurs (DeepMind Genie 3, Meta V-JEPA 2, World Labs Marble, NVIDIA Cosmos, Wayve GAIA-3, Tesla, 1X, AMI Labs, Runway, Decart), grand débat technique pixel vs latent, applications, et argument Hassabis sur la pièce manquante vers l'AGI.

## Comment ouvrir

- **Application illustrée (recommandé)** : ouvrir `20260505-world-models-app.html` dans un navigateur (double-clic). Aucun serveur requis. Sommaire à gauche, sources à droite (repliables sur desktop), 9 schémas SVG cliquables avec ~70 régions interactives, glossaire au survol des termes pointillés, plein écran via le bouton ⛶ au survol de chaque figure.
- **Version texte** : `20260505-world-models-rapport.md` — markdown compatible Obsidian, citations en notes de bas de page, schémas embarqués via `![alt|width](images/...svg)`.
- **Hub d'entrée** : `index.html` — page de présentation du dossier avec les deux formats.
