# Outline — demonstration-formelle

## Thèse
La démonstration mathématique formelle (Lean) est le seul domaine où la récompense du raisonnement machine est **dense, automatique et incorruptible** : le noyau du vérificateur ne se laisse pas *hacker*. C'est le laboratoire pur du RLVR. Mais la médaille d'or de l'IMO 2025 a été gagnée en **langage naturel non vérifié** — preuve que le raisonnement machine est à une bifurcation entre la garantie formelle (étranglée par l'autoformalisation) et la généralité naturelle (sans garde-fou).

## Plan (7 sections + Sources)
1. **La bifurcation** — IMO 2024 (AlphaProof, argent, Lean) → IMO 2025 (Gemini Deep Think + OpenAI, or, NL non vérifié). Deux routes, même sommet. Pose la thèse du vérificateur incorruptible. [SCHEMA-01]
2. **Pourquoi Lean ne ment pas** — anatomie : énoncé Lean → tactiques → état de preuve → noyau de confiance (de Bruijn) → QED ; Mathlib. Contraste avec PRM/LLM-juge reward-hackable (renvoi process-reward-models). [SCHEMA-02]
3. **AlphaProof : AlphaZero rencontre le solveur** — autoformalisation (~1M problèmes → ~100M énoncés Lean), boucle RL AlphaZero, test-time curriculum, 3/5 problèmes, 28 pts, P6 résolu mais jours de calcul ; AlphaGeometry 2 (P4). [SCHEMA-03]
4. **Les open-source provers — la course du benchmark** — DeepSeek-Prover-V2 (88,9 % miniF2F, subgoal RL/GRPO, 49/658 Putnam), Goedel-Prover-V2 (90,4 % self-correction, 86 Putnam), Aristotle (Harmonic, IMO or formel). miniF2F sature → PutnamBench nouvelle barre. [SCHEMA-04] [SCHEMA-05]
5. **Le goulot d'étranglement : l'autoformalisation** — NL→Lean = le vrai mur : imports hallucinés, syntaxe mélangée, dérive sémantique ; rareté des données ; synthèse échafaudée + expert iteration ; ProofBridge, IndiMathBench. [SCHEMA-06]
6. **La médaille d'or sans garde-fou** — Gemini Deep Think/OpenAI 35/42 end-to-end NL ; OpenAI gradé en interne (3 médaillés), pas de vérif formelle ; verification-and-refinement pipeline comme pont. Risque du plausible-mais-faux (renvoi process-reward-models). [callout]
7. **Le nouveau métier du mathématicien** — Equational Theories Project (Tao, 22 028 942 edges, 4 694 lois, Lean, crowdsourcé) ; Tao évangéliste lucide. Trajectoires 2026-2028 : autoformaliseurs fiables, preuve-as-CI, verifier comme reward universel, fusion NL propose/Lean dispose. [SCHEMA-07]
