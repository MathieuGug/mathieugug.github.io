# Benchmarks agentiques contestés

Deep dive sur `evaluation-agentique` : pourquoi SWE-bench, GAIA et OSWorld ne mesurent plus la capacité, mais l'effort d'optimisation d'une équipe dédiée.

> 15 mai 2026 — Mathieu Guglielmino

## Contenu

- `index.html` — hub du dossier
- `20260515-benchmarks-contestes-app.html` — application compagnon interactive (rapport + 6 schémas cliquables + sources)
- `20260515-benchmarks-contestes-rapport.md` — rapport markdown sourcé (Obsidian-compatible)
- `images/` — 6 schémas SVG éditoriaux

## Pour relire localement

Ouvrir `20260515-benchmarks-contestes-app.html` directement dans un navigateur. Aucun build ni serveur requis (la lib `/assets/dossier-app.{js,css}` est résolue par chemin relatif depuis la racine du site).

Le markdown se relit dans Obsidian ; les images SVG sont référencées via la syntaxe `![alt|width](images/...)`.
