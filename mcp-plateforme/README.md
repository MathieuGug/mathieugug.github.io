# MCP, le HTTP des agents

> 8 mai 2026 — Mathieu Guglielmino

Deep research illustré sur le Model Context Protocol (MCP) : architecture, écosystème, surface d'attaque, comparaison avec function calling / OpenAPI / A2A, trajectoire 2026-2027.

## Contenu du dossier

- `index.html` — hub d'entrée du dossier (deux cartes : app, rapport).
- `20260508-mcp-plateforme-app.html` — application compagnon illustrée. S'ouvre par double-clic, opère depuis `file://`. Six schémas SVG inline avec régions interactives, sommaire, sources, glossaire au survol, zoom plein écran.
- `20260508-mcp-plateforme-rapport.md` — rapport complet en markdown. Lisible dans Obsidian, VS Code, GitHub. Citations en notes de bas de page, schémas référencés en syntaxe Obsidian (`![alt|width=1200](images/...)`).
- `images/` — six SVG éditoriaux (anatomie protocole, timeline d'adoption, composition multi-server, surface d'attaque, comparatif des couches d'intégration, trajectoire 2026-2027).

## Comment lire

Pour la lecture interactive (recommandée) : ouvrir `20260508-mcp-plateforme-app.html` dans un navigateur. Cliquer les régions colorées des schémas pour les détails, survoler les termes pointillés pour les définitions, cliquer les `[N]` pour voir la source dans le panneau de droite. Bouton ⛶ au survol de chaque schéma pour zoomer en plein écran.

Pour la lecture textuelle : ouvrir `20260508-mcp-plateforme-rapport.md` dans Obsidian (les schémas s'affichent inline) ou tout éditeur markdown.

Format co-écrit avec l'aide d'une IA.
